The Stargate 323 is a mono to stereo reverb, so make sure to sum stereo input signals together. An FX-chain example for REAPER is included.
All IRs are 96Khz/32bit and with 0 PreDelay.
The naming scheme consists of the program number and decay time in seconds: pg4_1-8s = Program 4 & 1,8 seconds decay time.
